如何用hugo搭建个人博客

May 14, 2020

如何用hugo搭建个人博客(Windows).

所需软件(环境变量需要配置)

  • cmder
  • vscode

所需知识

  • linux命令行基本操作
  • get命令行基本操作

搭建步骤

  1. 在硬盘中创建用来存放项目的目录，路径不要带有中文，如图

  2. 安装hugo 去hugo的Github托管中心下载hugo_xxx_Windows-64bit.zip,解压后放到任意路径下(路径不能有中文),解压后添加环境变量。打开cmder或者cmd输入hugo version查看版本。出现版本号为正常，如果没出现，可以换低一点的版本下载。

  3. 进入hugo官网,点击quickstart，从step2开始，把各个步骤的命令行一次在cmder中输入 注意点：step2的hugo new site quickstart中的需要换成github名字(小写).github.io-creator。此步完成之后，会在你的项目文件夹中创建一个博客生成器。 双击进入博客生成器目录，此时在cmder中输入code开始VScode编辑器，之后的hugo命令，在VScode中新建终端来输入。

  4. 执行到step5,hugo server -D,之后会出现如下信息，复制红框标记连接到浏览器，即可进行访问。

  5. step6的时候需要变更配置信息 修改为

    baseURL = "https://你的GitHub的名字.github.io/"
languageCode = "zh-Hans"
title = "你的博客的名字"
theme = "ananke"

  6. 执行完step7后，会在生成器目录中，生成博客的部署项目文件夹public，此文件夹就是要部署到github上的内容。进入改目 录，用命令行将真个目录内容提交到本地仓库。 所用命令行

    git init
git add .
git commit -m "注释信息"

  7. 在github上创建仓库，用来存放该项目。创建过程如下

    • 这里只用填写红框的地方，其他地方默认

    • 在vscode中执行下图红框的命令行，上传至github

    • 刷新后会出来目录结构，然后点击Settings，下拉找到github pages，会显示your site is published at URL,点击URL就可以访问到博客了。

  8. 以上步骤为博客的上传，还需要将博客生成器的代码也进行上传。上传步骤同以上一样，只是需要注意两点。

    • 上传的时候一定要切换会博客生成器的目录，而不是在public目录中进行操作。
    • 在博客生成器目录中，需要重新创建git本地库，进行commit，在上传到github。此时需要把public目录排除，只需要创建如下文件，就可以排除public目录
